## BUSINESS LOGIC (MUST BE EXACT)

### TDS Calculation Function (pseudocode, implement in TypeScript)
Input: employeeId, fiscalYear, projectedAnnualGross (from current salary structures * 12), actualDeductions (SSF employee, insurance, etc.)

1. Determine taxable income = projectedAnnualGross - deductions.
2. Fetch taxSlabs JSON from fiscalYear record. Slabs are like:
   [
     { "limit": 500000, "rate": 0, "type": "tax" },
     { "limit": 200000, "rate": 10, "type": "tax" },  // next 200k at 10%
     { "limit": 300000, "rate": 20, "type": "tax" },
     { "limit": 1000000, "rate": 30, "type": "tax" },
     { "limit": null, "rate": 36, "type": "tax" }  // remaining
   ]
   Additionally, include a slab for 1% Social Security Tax (SST) on first 500,000, but note it is rebatable. For simplicity, the final tax calculated from the above table is the net tax. The SST is not separately deducted; the table given already accounts for it (as per current Nepali practice). So just apply the progressive rates.
3. Calculate annual tax: for each slab, apply rate to the portion of taxable income within that limit.
4. Monthly TDS = annualTax / 12. (Use simple projection; if employee joined mid-year, we can still project full-year income and deduct equally each month, as per IRD guidelines.)
5. Return monthly TDS amount.

Important: This computation must be correct for all income levels. Test with:
   - Taxable income 400,000 => tax 0 (only 1% SST but rebated, effectively 0)
   - Taxable income 700,000 => 10% on (700k-500k) = 20k
   - etc.

### SSF Contribution Calculation
- employeeSsf = basicSalary * fiscalYear.ssfEmployeeRate
- employerSsf = basicSalary * fiscalYear.ssfEmployerRate
- Split into sub‑heads according to percentages defined in fiscalYear for medical, pension, etc. (store these sub‑amounts in separate PayrollTransaction rows with isEmployerContribution flag where needed)
- Both contributions must appear on payslip and SSF report.

### Overtime Calculation (if overtime hours are entered)
- Hourly rate = (Basic + Dearness Allowance) / 240  (assuming 8hrs/day, 30 days or 26? Labour Act: 8hrs/day, monthly salary / 240 for hourly). Use formula: (Basic + DA) / 240.
- Overtime amount = hourlyRate * 1.5 * overtimeHours.
- Validate against max daily/weekly limits and warn.

### Festival Bonus (Dashain)
- Automatically computed in payroll run for periods covering Asoj/Kartik: add one month's Basic salary as a separate earning head.

### Final Settlement (on deactivation)
- Unpaid salary up to last working day.
- Leave encashment: for each unused annual leave (max allowed accumulation), daily rate * days.
- Gratuity: For service >= 1 year, calculate as per Labour Act: 15 days' basic salary for each completed year of service (or fractional part). Daily rate = (last drawn basic salary / 26). Total gratuity = (last basic / 26) * 15 * years. Note: if employee is in SSF, gratuity may be covered by SSF pension – but still compute as per law; the system should just compute for settlement report. No automatic payment via app.
- Deductions for outstanding loans/advances (just a manual deduction entry).
